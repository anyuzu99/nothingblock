# Tools used to build the project

## Header

``` adp
[Adblock Plus 3.0]
! Title: Nothingblock filter
! Homepage: https://github.com/sayo-melu/nothingblock
! Expires: 7 days
! Version: 2019.1.22
```

## URI

`abp:subscribe?location=[url]&title=[name]`

## Regular Expression

- For `!`

    `!.*`

- For `[return]`

    `^(\t|\n|)*\n`
